Se connecter en tant que invite

- Request

Method :    POST

URL :    /

Header :

Content-Type : application/json

Body :

{
    "login":"invite"
}

- Response - 201

Header :

Content-Type : application/json

Body :

{
    "status": "success",
    "status_code": 201,
    "status_message": "[R201 REST AUTH] : Authentification OK",

}
